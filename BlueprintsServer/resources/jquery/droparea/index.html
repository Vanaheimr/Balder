droparea is a HTML5 drag and drop image file uploader jQuery plug-in and a php script for server-side.

I developed it to use on my Edirne project.

I've only tested it on Google Chrome (11.0.696.65) and Mozilla Firefox (3.6.17) and it works enough for me for now!

goker.cebeci, the developer

download or fork it on github

jquery plugin page


USAGE

<div class="droparea spot" data-width="460" data-height="345" data-type="jpg" data-crop="true" data-quality="60" data-folder="sample" data-something="stupid"></div>

<script>
$('.droparea').droparea({'post' : 'upload.php'});
</script>
